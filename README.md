# Flashcard App

A beautiful, interactive flashcard application built with vanilla HTML, CSS, and JavaScript. Perfect for studying and memorizing information with an intuitive interface and smooth animations.

## Features

- **Create & Manage Flashcards**: Add, edit, and delete flashcards with questions and answers
- **Multiple Decks**: Organize your flashcards into different decks by subject or topic
- **Study Mode**: Interactive study interface with card flipping animation
- **Navigation**: Move through cards with Previous/Next buttons
- **Shuffle**: Randomize card order for better learning
- **Progress Tracking**: See which card you're on and total cards in deck
- **Data Persistence**: All data is automatically saved to your browser's localStorage

## How to Use

### Getting Started

1. **Open the App**: Simply open `index.html` in your web browser
2. **Default Deck**: The app creates a "My First Deck" automatically when you first open it

### Study Mode (Default View)

1. **View Cards**: The current flashcard displays the question on the front
2. **Flip Cards**: Click anywhere on the card to flip it and see the answer
3. **Navigate**: Use the "Previous" and "Next" buttons to move through your cards
4. **Shuffle**: Click "Shuffle" to randomize the order of cards
5. **Track Progress**: See your progress at the bottom (e.g., "Card 3 of 20")

### Manage Cards

1. **Switch to Manage Mode**: Click the "Manage Cards" button at the top
2. **Add New Cards**:
   - Enter your question in the "Question" field
   - Enter the answer in the "Answer" field
   - Click "Add Card"
3. **Edit Cards**: Click the "Edit" button on any card to modify it
4. **Delete Cards**: Click the "Delete" button to remove a card

### Managing Decks

1. **View Current Deck**: See the selected deck in the dropdown at the top
2. **Create New Deck**:
   - Click "+ New Deck"
   - Enter a name for your deck
   - Click "Create"
3. **Switch Decks**: Use the dropdown menu to select a different deck

## Tips for Effective Studying

- **Use the Shuffle feature** to avoid memorizing cards by order
- **Flip each card yourself** before checking the answer to test your recall
- **Create multiple decks** for different subjects to stay organized
- **Keep questions clear and concise** for better retention
- **Review regularly** to reinforce learning

## Technical Details

- **No Installation Required**: Just open the HTML file in any modern browser
- **Works Offline**: All data is stored locally in your browser
- **No Backend Needed**: Pure client-side application
- **Responsive Design**: Works on desktop and mobile devices

## Browser Compatibility

- Chrome (recommended)
- Firefox
- Edge
- Safari

## Data Storage

All flashcard data is stored in your browser's localStorage. This means:
- ✅ Your data persists between sessions
- ✅ No internet connection required
- ⚠️ Data is specific to the browser you're using
- ⚠️ Clearing browser data will delete your flashcards
